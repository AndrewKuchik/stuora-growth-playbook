# Stuora — Placement & Growth Playbook

Одностраничный сайт-навигатор: где коктейль-бару **Stuora** (Рига, Audēju iela 15) размещаться, чтобы привлекать туристов и местных — что каждый канал даёт, сколько стоит и как подать заявку в один клик.

🔗 **Живая страница:** https://andrewkuchik.github.io/stuora-growth-playbook/
✉ **Письма и заявки:** https://andrewkuchik.github.io/stuora-growth-playbook/letters.html

## Что внутри
- `index.html` — сам сайт (открывается двойным кликом или по ссылке GitHub Pages).
- `letters.html` — готовые письма и заявки: проверенные адреса, кнопки «копировать», ссылки на формы.
- `OUTREACH-KIT.md` — реестр контактов, правила языка обращений и журнал проверки каналов.
- `ARCHITECTURE.md` — как устроено, как править контент и **как добавить язык**.
- `Stuora-plan-prodvizheniya.md` — полный текстовый план продвижения (первоисточник контента).

## Как обновить
1. Открыть `index.html`, поправить нужное (карточки — в массиве `CH`).
2. Сохранить, затем:
   ```
   git add -A
   git commit -m "обновил план"
   git push
   ```
3. Через ~минуту страница обновится сама.

## Стек
Чистый HTML/CSS/JS без зависимостей · хостинг GitHub Pages (бесплатно) · светлая/тёмная тема · готовность к нескольким языкам.
