# Stuora (Рига) — финальный план продвижения

**Бар:** Stuora — небольшой атмосферный коктейль-бар, Audēju iela 15, Старый город Риги (вход с Rīdzenes iela). Тихая улочка в стороне от основных потоков, неприметный вход в духе speakeasy. Намеренно негромкое место: сюда приходят посидеть и нормально поговорить. Главная фишка — домашние настойки на натуральных сезонных ингредиентах; есть вегетарианские позиции и полностью безалкогольные версии напитков. Чт–Вс — walk-in, Пн–Ср — только приватные события. Instagram [@stuora.bar](https://www.instagram.com/stuora.bar/).

> Позиционирование обновлено владельцем (июль 2026): «рюмочная», винтаж и советские отсылки больше не используются ни в каких текстах.
**Аудитория:** туристы + местные (в т.ч. русскоязычные Риги) примерно поровну.
**Логика бюджета:** сначала выжать бесплатное, потом точечно платное.
**Главный принцип этого плана:** приоритет — площадки, **где бара ещё НЕТ** и где **большой трафик**.
**Дата:** июль 2026. Цены и составы подборок меняются — перепроверять перед действием.

# ПРИОРИТЕТ А. Где вас ещё НЕТ и где большой трафик (делать в первую очередь)

### А1. TripAdvisor — бесплатно ⚠️ №1 ПРИОРИТЕТ
- **Трафик/аудитория:** платформа №1 для туристов, планирующих поездку. Вас там нет — самый большой упущенный канал.
- **Цена:** создание и claim карточки — 0 € (платная только опциональная реклама).
- **Как попасть:** «Get Listed» → добавить заведение (название, адрес Audēju 15, тип, часы, бар/кухня) → «Claim Your Listing» → верификация → фото, меню, описание на EN.
- **Как стать рекомендованным (Travelers' Choice):** награду **нельзя купить/подать заявку** — автоматическая. Условия: карточка живёт ≥12 мес + порог отзывов за период + высокий рейтинг. Победители = топ‑10% (Best of the Best = топ‑1%). → **чем раньше создать и копить отзывы, тем раньше награда.** Вежливо просить довольных гостей-иностранцев оставить отзыв.
- Источники: [настройка](https://www.tripadvisor.com/business/insights/tripadvisor-listing-setup-restaurants), [Travelers' Choice](https://tripadvisor.mediaroom.com/travelers-choice)

### А2. Официальные тур-порталы Латвии — бесплатно, высокий авторитет
**LiveRiga** (портал Риги, ведёт муниципальное агентство туризма) и **latvia.travel** (нацпортал, госагентство LIAA) — у обоих есть кураторские коктейль-гиды. В [гиде latvia.travel](https://www.latvia.travel/en/where-enjoy-cocktail-riga-and-near-it) сейчас 10 баров, **Stuora нет**, а speakeasy nosaints — есть (значит формат подходит).
- **Трафик/аудитория:** туристы + вес «рекомендовано госпорталом».
- **Цена:** 0 €.
- **Как попасть:** (1) базовый листинг — e-услуга LIAA «Publishing information in latvia.travel» / письмо на **tourism@liaa.gov.lv**; LiveRiga — через контакты портала / Турцентр в Ратуше. (2) В РЕДАКЦИОННЫЙ гид — отдельный питч редактору: чем Stuora уникален (домашние настойки, «tomato & sprats Mary», рюмочная-концепт), 3–5 фото, ссылки.
- Источники: [LiveRiga Eat&Drink](https://www.liveriga.com/en/4-eat-drink), [e-услуга LIAA](https://www.liaa.gov.lv/en/services/publishing-information-latviatravel)

### А3. Atlas Obscura — бесплатно (неочевидно, но точное попадание)
- **Трафик/аудитория:** международный сайт про «скрытые, удивительные места»; любознательные путешественники. Есть рубрика Speakeasies. Скрытый бар с рюмочной-эстетикой — прямое попадание.
- **Цена:** 0 € (пользовательская подача).
- **Как попасть:** через [FAQ / Add a place](https://www.atlasobscura.com/faq) описать место: адрес, фото, «самое интересное» первой фразой (скрытый вход, домашние настойки, история концепта). Коротко и с «вау».

### А4. Платформы впечатлений — превратить бар в бронируемый «опыт» 💡 (большой турпоток)
Ключ: тут нельзя разместить «просто бар» — только **бронируемый опыт** (коктейльный мастер-класс / дегустация настоек). Бонус — заполняет пустые Пн–Ср.
- **GetYourGuide** — 40M+ визитов/мес. Размещение 0 €, комиссия только с броней (для Латвии — после регистрации, ориентир ~20–30%). Регистрация: [getyourguide.supply](https://www.getyourguide.supply/) → листинг Workshop/Activity «Cocktail-making masterclass at a hidden Riga speakeasy».
- **Viator** (подтягивается на Tripadvisor) — та же модель; в Риге уже есть коктейль-мастерклассы. [Пример](https://www.viator.com/Riga-tours/Food-and-Drink-Classes/d4480-tag21482)
- **Airbnb Experiences** — аудитория путешественников; в Риге уже есть коктейльные experiences.
- **byFood / Withlocals** — нишевые платформы впечатлений для доп. охвата.

### А5. Пабкраулы — трафик групп туристов «под ключ»
- **Операторы:** Riga Pub Crawl (×2), **Red Fox Tours** (маршрут Doma / St. Peter's / Peldu / Kaļķu — **ровно ваш квартал**, €25/чел).
- **Цена для бара:** обычно бесплатно (даёте welcome-drink — оператор приводит группу).
- **Как попасть:** формальной заявки нет — только прямое письмо (напр. info@riga-pub-crawl.com). Питч: «интимный speakeasy как необычная точка маршрута, welcome-shot на домашней настойке».
- ⚠️ Для интимного формата брать малые/премиальные краулы, не шумные толпы.
- Источники: [rigapubcrawl.com](https://rigapubcrawl.com/), [redfoxtours.eu](https://redfoxtours.eu/riga-pub-crawl.html)

### А6. Apple Maps + Bing Places — бесплатно (быстро, большой охват)
- **Apple Maps (Apple Business Connect)** — туристы с iPhone (США/ЕС) ищут «bar near me» именно тут. Вас там, скорее всего, нет. Зарегистрировать/подтвердить — 0 €.
- **Bing Places** — 🔧 уже настраиваете; питает Bing и ассистента **Copilot**. Довести до конца.
- **Foursquare** — бесплатно; данные питают множество тур-приложений и city-guides.

### А7. Международные редакционные фичеры — бесплатно (заработанный PR)
Издания, которые УЖЕ пишут про рижские бары — питчить с уникальным углом:

# ✅ План действий по приоритету

### Неделя 1 (бесплатно, максимальный эффект — «где нас нет + большой трафик»)
1. **Создать TripAdvisor** (Get Listed + claim) — крупнейший пробел.
2. **Подать в Atlas Obscura** — идеальный фит для скрытого бара.
3. **Питч-письма** редакциям LiveRiga и latvia.travel (tourism@liaa.gov.lv) — в коктейль-гид.
4. **Доделать Bing Places**, завести **Apple Maps** + **Foursquare**.
5. Оптимизировать **Google Business Profile** (фото, категории, QR на отзывы).

### Недели 2–4 (бесплатно/партнёрско)
6. Обойти консьержей (упор — отели без бара) — карты + welcome-drink.
7. Письма 3 операторам пабкраулов (Red Fox и др.).
8. Партнёрство с **Escape Room Riga** + корпоративы Пн–Ср.
9. Питч в LIAA на блог-тур + 3–4 изданиям (Vogue Scandinavia, Lonely Planet, Neighborhood.lv).
10. Найти 5–10 локальных микро-блогеров (в т.ч. русскоязычных) на бартер.
11. Завести карточки в локальных каталогах (1188, Gurmania, Nomax, Wanderlog, LikeALocal, Local Life, Riga Guide).

### Месяц 2+ (платно/вдолгую)
12. Оформить **коктейль-мастеркласс/дегустацию** → GetYourGuide + Viator + Airbnb (+загрузка Пн–Ср).
13. 2–3 Reels у рижских креаторов.
14. Регулярный **паб-квиз** → афиша Extrareality.
15. Ко-брендинг с брендом алкоголя (Black Balsam/локальная дистиллерия).
16. Отраслевые конкурсы (Monin Cup, Riga Food, Baltic Wine & Drinks) → радар Europe's 50 Best.

### По желанию (в любой момент)
- Забрать чужие авто-карточки — начать с **RestaurantGuru** (см. отдельный блок).
